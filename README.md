# preSpace

A minimalist, one-button drifting parking game.
Space key to start, Space key to drift.

## How to Play
- Press [SPACE] to start the car.
- Press [SPACE] again to drift into the parking spot.
- Release [SPACE] to stop drifting.
- Try to park perfectly in the designated spot!

## Controls
- **SPACE / TAP**: All actions
